Installation instructions

Before trying to install anATLyzer check that you have installed its dependencies:

  • ATL (from version 3.0) - http://www.eclipse.org/atl/

You can use the update site or you can download the packaged update site from one of the links at the left sidebar. Install with:

With update site: Help -> Install New Software -> Add... -> Paste the URL
Packaged site: Help -> Install New Software -> Add... -> Archive ...

It may be needed to uncheck "Group items by category" if nothing appears in the plug-in list.
